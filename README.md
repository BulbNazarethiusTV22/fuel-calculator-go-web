# Калькулятор палива — Go Web

## Запуск
1) Встанови Go (1.20+ бажано)
2) У папці проекту:
```bash
go run .
```
3) Відкрий у браузері:
- http://localhost:8080

## Функціонал
- **Завдання 1**: перерахунок складу на суху та горючу масу + розрахунок Qрн/Qсн/Qгн
- **Завдання 2**: мазут — перерахунок на робочу масу та Qрн

## Примітки
- Десятковий роздільник: крапка або кома.
